# ReviewPal Demo (Rust)

A simple Rust project to demonstrate [ReviewPal](https://github.com/Arephan/reviewpal) AI-powered code reviews.

## What is ReviewPal?

ReviewPal automatically reviews your pull requests using Claude AI, catching:
- 🔒 Security vulnerabilities
- 💥 Potential crashes
- 🗑️ Data loss risks  
- 🐌 Performance issues

## Setup

1. Add `ANTHROPIC_API_KEY` to your repository secrets
2. Create PRs and watch the magic happen!

## Building

```bash
cargo build
cargo run
```
